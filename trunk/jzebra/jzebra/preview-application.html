jZebra Applet




To view the applet's html source code: Right Click This Page --> View Source and look for <applet> </applet> code.

To get the latest version, or submit a bug visit: http://code.google.com/p/jzebra


Config Error: The applet will fail to load the config file until it is place in C:\ or the applet param "config" has an appropriate value. Use absolute paths such as "file:///C:/preview-raw-commands.txt" if unsure. UNIX users can specify "file:///home/user..." instead. Web developers can use "http://site.com/..."

Message Dialogs: Error and success messages support <html></html> formatting within param tags. Message title parameter (messagetitle) does not support html formatting within tag. These message values are only displayed if applet param "showmessages" is set to "true".

Java Console: Please copy/paste details from the java console when submitting a bug report
